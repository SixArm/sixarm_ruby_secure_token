Random token generator to create strong secure text tokens.

Example

  SecureToken.new => "kavzwbnxremyqlkw"

This generates a 16-character token of all lowercase letters, using Ruby’s securerandom methods.

SecureToken is a string, so you can do any string methods on it. You can change how tokens are randomly created, however you want.

SecureRandom

Ruby 1.8.6 and older does not have a secure random number method, so this gem checks to see if the SecureRandom class is defined; if it is not, then we require our sixarm_ruby_secure_random gem.
